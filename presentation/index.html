Hello Elixir


iex(1)> IO.puts "Hello Elixir"
Hello Elixir
:ok

iex(2)> add = fn a, b -> a + b end
#Function<12.118419387/2 in :erl_eval.expr/5>
iex(3)> add.(2,3)
5

iex(4)> events = [:create, {:name, 'Ben'}, 200, "dogs"]
[:create, 100, 200, "dogs"]

iex(5)> [head | tail] = events
[:create, 100, 200, "dogs"]

iex(6)> head
:create

iex(7)> tail
[100, 200, "dogs"]
* 5+ Years Old * Created to be a more productive way to leverage concurrency ** Ruby like syntax ** Functional ** Dynamic Typing ** Pattern matching ** Equals is equality statement more than assignment * Built on the Erlang VM ; influenced by Ruby * Erlang - telcom, high avail, concurrency

"Bob"
* All about the FAST, light weigh, concurrency * Message passing, no shared state * Clean productive syntax without a lot of ceremony * CODE * - spawn takes a fun and runs concurrently to current process * - process runs until it finishes the fun and then dies * - process != OS process * - processes not just for one off tasks, can send and receive msgs
